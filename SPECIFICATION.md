# Simple C++ Specifications

## 1. Scope

This document defines the **formal behavior and semantics** of the Simple C++ language.

It includes:
- type system
- casting rules
- operator behavior
- language constructs

This document is implementation-oriented and intended to support parser and transpiler development.

---

## 2. Type System

### 2.1 Overview

The Simple C++ type system defines a restricted and controlled set of types
implemented via a runtime library.

All user-visible types are mapped to wrapped C++ types under the `scpp` namespace.

Runtime types in the `scpp` namespace are not user-facing and are generated by the transpiler.

All values in Simple C++ must conform to the internal type system.

---

### 2.2 Primitive Types

The following primitive types are supported:

- `bool`
  - Values: `true`, `false`
  - Backed by: `scpp::bool_t`

- `int`
  - 8-byte signed integer
  - Backed by: `scpp::int_t` (`long long`)

- `float`
  - 8-byte floating point
  - Backed by: `scpp::float_t` (`double`)

Primitive types are value types:
- passed by value (copied)
- returned by value
- not heap-allocated

---

### 2.3 Standard Library Types (Managed)

The following managed wrapper types are provided:

- `string`
  - Wrapper over `std::string`
  - Type: `scpp::string_t`

- `vector<T>`
  - Wrapper over `std::vector<T>`
  - Type: `scpp::vector_t<T>`

Where:
- `T` must be a valid Simple C++ type or user-defined type

Additional types may be introduced in future versions.

---

### 2.4 Passing Semantics

#### Default Rules

- Primitive types:
  - passed by value
  - returned by value

- `string`, `vector<T>`:
  - passed by `const &` (to avoid copies)

- Other types:
  - should be wrapped in:
    - `shared_p`
    - `unique_p`
    - `weak_p`
  - passed by value (pointer semantics)

#### Explicit References

Explicit pass and return by reference is allowed.

---

### 2.5 Composite Types

#### `null_t`

Simple C++ defines a universal null type:

```cpp
struct null_t {
	constexpr operator std::nullptr_t() const { return nullptr; }
	constexpr operator std::nullopt_t() const { return std::nullopt; }
};

inline constexpr null_t null{};
